
# 🚗 Car Data SQL Analysis

This project contains SQL queries performed on a car dataset.

## 📂 Dataset
- CarDekho dataset (CSV format)

## 🛠️ Tools Used
- MySQL / PostgreSQL

## 📊 SQL Concepts Covered
- SELECT, WHERE
- GROUP BY
- Aggregate Functions
- Filtering by year
- Finding max/min values

## 📌 Sample Questions Solved
- Total cars available per year
- Year with more than 10 cars
- Second highest price car
- Cars between 2015–2023
